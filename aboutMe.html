About me and this page

我是陈鑫，一个不怎么喜欢发朋友圈的家伙，欢迎来到我的主页！
我对花里胡哨的网页排版不怎么感兴趣，所以在编写网页的过程中，我尽量追求只使用html/css语言。 但是因为懒的原因，在写博客的时候还是基于 Hugo 框架来完成的。➝__➝ My blog
具体来说，我拥有两个Github账号，一个用来生成我的 主页 ， 而另一个用来专名写 博客 ，这两种都是静态网页。 其中主页是我自己用纯 html 和 css 语言编写的，我可以自由控制该主页的样式，您可能认为我的主页有点丑或土，但在我看来有种复古美。 而博客页面使用了 Hugo Hugo框架，用现成的网页主题，这样做的好处是， 我可以专注地用 markdown 语言来写博文的内容，然后用 git 发布到 github page 的页面上。 或许以后有空了会具体写一下这两种方案的实现细节（也可能永远没有时间）。
总之，希望这些网页上的内容能对 你/您 有用，祝您生活愉快🤪！
